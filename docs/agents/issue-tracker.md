# Issue tracker

Issues are tracked in GitHub Issues for this repo (`bioShaun/persona_seq`).

## Creating an issue

Use `gh issue create` from the repo root.

## Required fields

- Title
- Body (markdown)
- Labels (optional, but preferred for triage)

## Workflow

1. Create issue with `needs-triage` label
2. Triage to one of: `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`
3. Close when resolved
